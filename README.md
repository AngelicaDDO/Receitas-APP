# Receitas-APP
## Requisitos Funcionais
* Autenticação de usuário com bearer
* Cadastro de usuário
* Recuperação de usuário
* Criação de receita
* Remoção de receita
* Edição de receita
* Listagem com paginação de receita
* Página única de receita

## Requisitos Não Funcionais
* SQL Server
* Windows
* VSCode

## Linguagens
* C#
* AngularJS

## Tecnologias
* Aspnet Core
* Entity Framework
* AngularJS
